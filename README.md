## Code Examples
How to create a struct in solidity:

` struct Zombie {
        string name;
        uint dna;
    }`
     
  How to declare an event:
  
  `event NewZombie(uint zombieId, string name, uint dna);`
  
  How to fire the event
  
  ` emit NewZombie(id, _name, _dna);`
